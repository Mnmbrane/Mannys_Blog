# project_blog

A GitHub Pages blog powered by Jekyll.

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Push this folder to the repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Build and deployment` source to `Deploy from a branch`.
5. Choose branch `main` and folder `/ (root)`.
6. Save, then wait for the deploy.

Your site URL will be shown on the Pages settings screen.

## Local edits

- Blog posts go in `_posts/`
- Filename format: `YYYY-MM-DD-your-title.md`
- Pages like About go in root markdown files such as `about.md`

## Customize

Edit `_config.yml`:

- `title`
- `description`
- `author.name`
